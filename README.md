# String Between

Function to find String Between "Hello" and "World"

### **Input Format**

A single line containing string _**s**_.

### **Constraints**

Input contain String "Hello" and "World", respectively.

### **Output Format**

String between "Hello" and "World"

### **Sample Input**

```
"Hello 1234 World"
```

### **Sample Output**

```
"1234"
```
